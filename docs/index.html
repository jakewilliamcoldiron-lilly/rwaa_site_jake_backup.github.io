Welcome to the RWAA Statistical Analysis Team Onboarding Guide.

This guide helps new analysts set up systems, access data environments, and complete required training.

Start Here

New members should begin based on their employment type.

Lilly Full-Time Employees (FTEs)

👉 Start with
Common Foundation → Step 1 – Identity Manager

Lilly Vendor / Contractor

👉 Start with
Common Foundation → Step 2 – CLUWE Access

The onboarding sequence differs between FTEs and Vendors.

See Key Differences at a Glance for a detailed comparison of the two tracks.

Overview

This guide is divided into two parts:

Part A – Common Foundation

Core steps all members must complete.

Part B – Specialization Tracks

RWD · HTA · GPS · HEOR specific steps.

Two distinct onboarding tracks

Lilly FTEs and Vendors (contractors) follow materially different processes across most steps.

Key differences include:

  • System access

  • Step order

  • Training platforms

  • Tools

  • Time entry systems

Each step includes separate FTE and Vendor tabs.

Always follow the tab matching your employment type.
